# Como usar

## 1 passo
### clonar repositório
```bash
git clone https://github.com/henriquepython/nestwithgraph.git
```

```bash
## entrar pasta projeto
cd myapp
```


## 2 passo

```bash

# instalar dependências
npm install

# executar o projeto
ng serve

# Acesso ao playground
http://localhost:4200/
```
