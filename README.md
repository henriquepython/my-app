# Como usar

## 1 passo
### clonar repositório
```bash
git clone https://github.com/henriquepython/my-app.git
```

```bash
## entrar pasta projeto
cd my-app
```
## 2 passo

```bash

# instalar dependências
npm install

# executar o projeto
ng serve

# Acesso
http://localhost:4200/
```
